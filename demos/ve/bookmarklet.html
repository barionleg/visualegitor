Bookmarklet for side-loading: ve.collab

Usage

  • Host demos/ve/ve-collab-sideLoad.js on localhost:8000 (or change the URL in the script above)
  • Drag the link above to the browser bookmark bar, to create a ve.collab bookmarklet
  • Open a wiki page e.g. this one
  • In that tab, click the ve.collab bookmarklet to start
